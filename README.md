# Section Library

A growing collection of reusable UI sections — each section lives in its own folder with complete HTML, CSS, and JavaScript. Copy any folder into your project, swap the content, and it's ready to use.

## Sections

| Folder | Description |
|---|---|
| [product-card](./product-card/) | Glass product card with wishlist button, image hover swap, INR price, Add to Cart |
| [add-to-cart](./add-to-cart/) | Horizontal product card with quantity controls, live price update, stock badge |

---

## How to Use Any Section

1. Copy the section folder into your project
2. Open `index.html` to see the section standalone
3. Edit the content (images, text, price) as needed
4. Link the `style.css` and `script.js` in your main project

Each section folder has its own `README.md` with full usage instructions.
